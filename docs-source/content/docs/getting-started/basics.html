Just edit a source file and reload the page. To see this in action, lets make a few small changes to the stock Sate boilerplate.

Open the website.json file in your favorite editor and take a look at it's structure. This file acts as the "manifest" for your Sate website. The website.json file contains all the site-wide configuration and maps the relationships between pages on your site.

For now, find the value under the key pageDefaults.title and change the string "Your New Sate Site" to the name of your own website. Save the file, and reload the browser page. You should see the new page title show up as the page title. For example, in Chrome this is displayed in the browser tab for the page.
